[9095-합 경우의 수 점화식 도출 과정]

- 4의 경우;

…, +1
…, +2
…, +3

즉, 모든 경우는 반드시 아래 중 하나다:

(n-1을 만든 모든 경우) + 1

(n-2을 만든 모든 경우) + 2

(n-3을 만든 모든 경우) + 3

이렇게 "마지막 숫자를 기준으로" 그룹을 나눌 수 있다.

모든 경우 = 마지막이 1인 경우의 수 + 마지막이 2인 경우의 수 + 마지막이 3인 경우의 수..즉,
```java
f(n) = f(n-1) + f(n-2) + f(n-3)
```
