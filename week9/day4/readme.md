[DFS]

- "가능한 모든 경로의 수" = DFS
  - 최초 경로길이를 1로 잡을 것인지, 0으로 잡을 것인지 두가지 경우 모두 확인 필요하다.
    - dfs(start, 1); 
    - dfs(start, 0);
  - 백트래킹을 고려해서, 최대한 백트래킹을 고려하지 않아도 되도록 매개변수로 넘겨주는 동시에 증감연산 할 것
    - visited[nx][ny] = true;
      dfs(new int[]{nx,ny}, dist + 1); //백트래킹을 위해 매개변수 증감은 넘겨주면서 동시에
      visited[nx][ny] = false;

- 던전
  - 요소의 개수도 작고, 모든 경로를 탐색해야 최대로 지나갈 수 있는 던전의 개수를 추출할 수 있다.
